## What I'm working on

### Keysyntax

A typing game for people learning programming.
It helps users practice typing real code, including symbols, brackets, and programming syntax.

Main features:

* Code-based typing practice
* Typing speed and accuracy tracking
* CPU battle feature
* Online battle feature

### Marugoto Festival App

A web app concept for supporting visitors and organizers at a school festival.
The goal is to help people check schedules, maps, announcements, and useful event information smoothly.

### Database Design Documentation Tool

A tool for designing and sharing database structures.
It helps developers organize tables, columns, relations, and project documentation in one place.

### Small Web Tools

I also make small web apps and experiments, such as games, UI prototypes, and automation tools.
